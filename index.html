This site documents the three essential html elements needed to create a table on a web page.

Element Opening Tag Closing Tag Purpose
Table <table> </table> Creates a Table container element.
Table Row <tr> </tr> Creates a Table Row container element.
Table Data <td> </td> Creates a Table Data container element. This is sometimes also called a 'cell'.

Note to dev: see Cc HTML TABLES 12/13 for their css file. You can try it later to see if it works for you.
